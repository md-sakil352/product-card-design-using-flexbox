# 🛍️ Product Card Design using Flexbox

A clean and modern **Product Card UI Design** built using **HTML5 and CSS3 (Flexbox)**.  
This project demonstrates how to create responsive and visually appealing product cards using Flexbox layout.

## 🌐 Live Demo
👉 https://md-sakil352.github.io/product-card-design-using-flexbox/

## 📌 Features
- Modern and clean product card design  
- Built with CSS Flexbox  
- Responsive layout for different devices  
- Simple and well-structured code  
- Easy to customize  

## 🛠️ Technologies Used
- HTML5  
- CSS3 (Flexbox)  

## 🎯 Purpose
This project was created to practice **Flexbox layout system** and improve front-end design skills by building a real-world UI component.
